# Instalacion 

```
npm i --save @nestjs/passport passport passport-jwt
npm i --save-dev @types/passport-jwt
```

# Arrancamos Proyecto

```
npm run start:dev
```

# Controllers & Services

```
nest g mo modules/auth
nest g res modules/auth
```
# Controller & Service

```
nest g co modules/auth
nest g s modules/auth
```
